# FamilyTree

##Overview

FamilyTree Vault is a comprehensive genealogy application designed to help families preserve their history securely and privately. Built with modern web technologies and infrastructure-as-code principles, it provides a robust platform for managing family relationships, stories, photos, and historical records.

##Key Highlights [Future Development] 

-Privacy-First: All data stored securely on your infrastructure
-Rich Visualization: Interactive family tree graphs and timelines
-Export Capabilities: GEDCOM and JSON export for data portability
-Revision History: Track changes and maintain historical accuracy
-Duplicate Merging: Intelligent deduplication of family records
-Shareable Branches: Share specific family branches while maintaining privacy

##Tech Stack

###Frontend

-HTML5: Semantic markup and structure
-CSS3: Modern styling and responsive design
-JavaScript (ES6+): Dynamic interactions and client-side logic

###Backend

-Python 3.12: Application logic and API
-Flask 2.0+: Web framework and routing
-SQLite: Embedded database for data persistence

###Infrastructure & DevOps

-Terraform: Infrastructure as Code for AWS EC2 provisioning
-Ansible: Configuration management and server setup
-GitHub Actions: CI/CD pipeline automation (if configured)
-Docker: Containerization for deployment

###Cloud Platform

-AWS EC2: Application hosting
-AWS Security Groups: Network security
-AWS IAM: Identity and access management
